# GitHub Actions CI/CD Pipelines

Complete CI/CD pipeline collection built with GitHub Actions.

## Workflows

### 1. Hello GitHub Actions
- Basic workflow triggered on push
- Prints message, date, and directory

### 2. Python Tests
- Automatically runs pytest on every push
- Tests add, subtract, multiply functions

### 3. Docker Build
- Builds Docker image automatically on push
- Runs tests inside container

### 4. Full CI/CD Pipeline
- Complete pipeline with 3 jobs
- test → build → summary
- Jobs run in order, stops if any job fails

## How it works
Push code to GitHub → Actions run automatically → Results in Actions tab!
